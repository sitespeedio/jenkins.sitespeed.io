Configure the limits for your JUnit/TAP or budget. The configuration is the same independent of the output. Read more about how to configure at the docs. Example setting budget for page assets: 
{
	"page": {
    	"image": 10,
        "css": 1,
        "requests": 7,
        "pageWeight":300000,
        "jsWeight": 0
    }
}
Example setting budget for timings: 
{
	"timings": {
    	"speedIndex": 1000,
        "headerTime": 800,
        "logoTime": 500,
        "domContentLoadedTime": 900
    }
}
Example setting budget for sitespeed.io rules (all rules needs to score 90 or more): 
{
	"rules": {
    	"default": 90
  	}
}
Example setting budget for WebPageTest: 
{
	"wpt": {
    	"requests": 60,
        "bytesIn": 1000000,
        "SpeedIndex": 1000
    }
}
You can of course also combine budgets.
